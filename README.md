## 📖 Project Overview

This project was created while learning **Onshape** during the **Smart Methods Cooperative Training**.

Starting from a reference image, I recreated the My Melody character as a layered 3D keychain using multiple sketches and extrusion operations. The final model is intended for visualization and can also serve as a base for future 3D printing.

## ✨ Design Workflow

The model was built using the following workflow:

1. Imported a reference image into a sketch.
2. Traced the outer outline using **Spline**.
3. Created a second sketch for the facial features and decorative details.
4. Extruded the main body **4.0 mm**.
5. Extruded the outline details **0.8 mm**.
6. Extruded the face, flower, flower center, and nose **0.5 mm** in the opposite direction to create a layered appearance.
7. Designed the keyring loop using concentric circles and attached it to the main body.
